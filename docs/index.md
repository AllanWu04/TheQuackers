# Project Overview
Our project aims to implement reinforcement learning to a Duckietown simulated environment, in which we can convert to a physical duckietown robot agent. 
In this project, we are trying Proximal Policy Optimization (PPO) and Soft Actor-Critic (SAC) algorithms to determine which would best help our driving agent
perform lane following. We hope to gain a deeper understanding of how to apply reinforcement learning algorithms to real-life applications and learning the 
intricate details of each algorithm to maximize the best results.
